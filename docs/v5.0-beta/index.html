📖 安装教程 · 小白版（一步一页 · 跟着翻就行）

1
下载安装包

点下方大蓝按钮，浏览器立即开始下载安装包到你的 「下载」文件夹。

⬇️ 一键下载 · 1.7 MB

💡 已下载过的可以直接点「下一步 →」

2
解压到 C:\AI-Arena ⭐

右键刚下载的 zip → "全部解压" → 在弹窗里粘贴推荐路径：

C:\AI-Arena

💡 解压完打开这个文件夹 · 能看到 manifest.json 就是对的

3
打开 Chrome 扩展页

打开 Chrome → 在地址栏粘贴这一行（复制后粘）：

chrome://extensions

回车 → 进入 扩展程序管理页。

💡 别用 360 / QQ 浏览器，要 Google Chrome 或 Edge。Edge 也支持，地址换成 edge://extensions

4
开启开发者模式 + 加载扩展
  1. 页面 右上角 打开「开发者模式」开关
  2. 页面 左上角 点「加载已解压的扩展程序」按钮
  3. 在弹窗中选刚才解压的 C:\AI-Arena 文件夹
  4. 看到 "AI Arena" 卡片出现 = 成功 ✓

⚠️ 报错"找不到 manifest.json" = 选错文件夹层级 · 应该直接选 C:\AI-Arena

5
登录 AI 网站，开始使用 🎉

扩展不存账号，对话走你浏览器里已登录的 AI 官网。至少登录 3 家：

🔵 DeepSeek 🥟 豆包 🐫 通义千问 🌙 Kimi 💰 元宝

🎉 完成！点 Chrome 工具栏的 AI Arena 图标即可使用
