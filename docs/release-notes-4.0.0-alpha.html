AI Arena v4.0.0-alpha 微信群聊视图

2026-05-22 · Phase 1（骨架 + 基础广播）

新增

  • 独立 popup 群聊窗口：点 sidepanel 顶部"🪟 群聊"按钮打开，类似微信电脑端
  • 多 AI 聚合时间线：3 个 AI 的回答以气泡形式落在同一页面，1.5 秒渐进刷新
  • @mention 单发：输入 @Claude 只发给 Claude，无 @ 则广播
  • 富文本兜底：检测到 Artifact / Canvas / Mermaid / 多图时显示"↗ 跳原页"按钮
  • 历史持久化：chrome.storage.local 保留最近 100 条，popup 重开自动恢复
  • 智能选屏：双屏环境下默认落在主屏右半，位置记忆

体验建议

  • 推荐启用 并列模式（3 个 AI 各自独立 window），群聊 popup 不会踩 Chrome 后台 tab throttling
  • 双屏：副屏铺 3 个 AI，主屏右半放群聊 popup

已知限制（Phase 2/3 解决）

  • sidepanel 同时提问 / 辩论 / PPT 工坊仍保留，未搬入 popup 抽屉 → Phase 2
  • 不拉取 AI 原页已有历史（白板起步） → v1.1 加折叠回溯
  • 富文本（Artifact / 图片）一律跳原页，不内嵌渲染 → v3+
