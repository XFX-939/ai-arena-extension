### 加载到 Chrome（4 步搞定）

```
① 解压下载的 zip（解压后能看到 manifest.json）
② Chrome 地址栏粘 chrome://extensions → 开启"开发者模式"开关
③ 点"加载已解压的扩展程序" → 选解压目录
④ 至少登录 3 家 AI 官网即可使用
```

💡 **图文版安装教程**：[📖 在线安装指引](https://tianlin0509.github.io/ai-arena-extension/docs/v5.0-beta/#install-guide)（含截图 + 9 AI 平台登录表）
